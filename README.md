## 🧪 About This Project

This portfolio demonstrates my ability to plan, design, execute, and document software testing activities on a real web application.

The **QACurry BookStore** is a browser-based e-commerce application that allows users to register, log in, browse books, add them to a cart, and place orders. It also exposes a REST API documented via Swagger.

**Testing performed:**
- ✅ Manual UI Testing — functional testing of all user-facing features
- ✅ API Testing — endpoint validation using Postman
- ✅ Bug Reporting — real defects found and documented with screenshots
- ✅ Automation Testing — key UI flows automated using Selenium WebDriver (JavaScript)
